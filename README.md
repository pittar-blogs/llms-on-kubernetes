# LLMs on OpenShift

This repository contains the code and configuration for the blog post: **[LLMs on Kubernetes: The Easy Way](blog/llms-on-k8s-the-easy-way.md)**.

It demonstrates how to easily deploy Large Language Models (LLMs) on Red Hat OpenShift using the Red Hat AI Inference Server (based on vLLM).

*   **[Blog Post](blog/llms-on-k8s-the-easy-way.md)**: The full technical guide.
